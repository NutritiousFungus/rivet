## üéØ USAGE NOTES FOR FIGMA

### **Creating a Folder Structure**
Use slashes (/) in style names to create organized folders:
- `Rock River/Primary Blue`
- `Rock River/Primary Yellow`
- `Dark/Background`
- `Light/Background`
- `Status/Complete Green`

### **Saving Color Styles**
1. Select any element with the color
2. In the Fill section, click the four-dot icon
3. Click the + icon
4. Name it exactly as shown above
5. Click "Create Style"

### **Saving Text Styles**
1. Select a text layer with the desired properties
2. In the Text section, click the four-dot icon
3. Click the + icon
4. Name it exactly as shown above
5. Click "Create Style"

### **Responsive Considerations**
- Desktop: 16px base font size
- Mobile: Same base, but some elements reduce (e.g., navigation)
- Breakpoint: 768px (md:) for most responsive changes
